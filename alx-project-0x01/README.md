# ALX Project 0x01 - Next.js Web Application

A comprehensive Next.js web application demonstrating fundamental concepts of modern web development.

## Features
- Multiple pages for displaying posts and users
- Interactive components for adding new content
- TypeScript for type safety
- Tailwind CSS for styling
- Component-based architecture

## Setup Instructions

1. Clone the repository
2. Navigate to the project: `cd alx-project-0x01`
3. Install dependencies: `npm install`
4. Run development server: `npm run dev`
5. Open browser to: http://localhost:3000

## Pages
- **Home**: Welcome page
- **Posts**: Display posts with mock data
- **Users**: Display users with mock data

## Technologies
- Next.js
- TypeScript
- Tailwind CSS
- React
